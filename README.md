# Freg-mq-metrics
Appen er datakjelde for [dokumentvakt-boardet Team Dokumentløysingar har i Grafana](https://grafana.nav.cloud.nais.io/d/a160bad7-da30-491a-b7e4-6449089741bb/dokumentvakt).
Boardet er særskild nyttig i oppgåvene som inngår i morgonvaktrutina til teamet.

Appen hentar data frå MQ-manager og rapporterer enkle metrikkar.

# Freg-mq-metrics
Appen er datakjelde for [backoutkø-boardet Team Dokumentløysingar har i Grafana](https://grafana.nais.io/d/gX0Fl-Nik/backout-koer?orgId=1).
Boardet er særskild nyttig i oppgåvene som inngår i morgonvaktrutina til teamet.

Appen hentar data frå MQ-manager og rapporterer enkle metrikkar.
